# 📦 Huffman File Compressor (C++)

A simple file compressor using the **Huffman Encoding algorithm** in C++.

## 🚀 Features

* Reads text from `input.txt`
* Generates frequency table
* Builds Huffman Tree
* Encodes data into binary
* Saves compressed output in `compressed.bin`

## ▶️ How to Run

```bash
g++ main.cpp -o compressor
./compressor
```

## 📁 Files

* `main.cpp` → source code
* `input.txt` → input file
* `compressed.bin` → compressed output

## ⚠️ Note

* No decompression yet
* Small files may not compress well

## 🧠 Concepts

Huffman Coding, Trees, Priority Queue, File Handling, Bit Manipulation
